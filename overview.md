# 🎓 UEMS - University Event Management System

A centralized platform to manage university events efficiently, connecting organizers and participants with role-based access and streamlined workflows.

---

## 🔹 Overview

UEMS allows admins and teachers to create, update, and delete events while students can browse and register for approved events. The system ensures:

- Role-based access control (Admin, Teacher, Student)
- Event creation, modification, and deletion by organizers
- Event search and filtering by name, date, or organizer
- Student registration and optional payment processing
- Tracking event status and organizer information

Designed to simplify event management and provide a seamless experience for both organizers and participants.

---

## 🚀 Key Features

- **Role-Based Management:** Different access and permissions for Admins, Teachers, and Students
- **Event Lifecycle:** Create, update, and delete events efficiently
- **Search & Filter:** Quickly find events based on organizer, name, or date
- **Registration & Payment:** Secure student registration and payment verification
- **Event Tracking:** Monitor status and registration progress

---

## 📌 Benefits

- Reduces manual coordination for university events
- Provides a clear overview of upcoming events
- Enables students to discover events organized by new or existing teachers/clubs
- Supports conditional actions like deletion only before event acceptance

---

## ⚡ Usage

- Organizers can manage events seamlessly
- Students can browse, filter, and register for events
- Administrators can oversee the entire system
