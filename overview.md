Project Overview

UEMS (University Event Management System)

UEMS is a system for managing university events. It allows admins and teachers to create, update, and delete events, while students can browse and register for approved events. The system supports role-based access, event filtering, and registration management, including payment verification for paid events.

Key functionalities include:

Role-based event management (Admin, Teacher, Student)
Event creation, updates, and deletion by organizers
Event search and filtering
Student registration and payment handling
Tracking event status and organizer information

It is designed to streamline event organization and provide a centralized platform for both organizers and participants.
